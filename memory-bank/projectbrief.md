# Aperture Leadership (apertureleadership.com)

Laravel app under `public_html/` with `index.php` at web root and application code in `public_html/application/`. Public gallery, user accounts, folders/cart, group chat scoped by `approval_code`, admin for albums/categories/images.

**Staging:** Planned at `stg.*` subdomain; category/photo work (Architecture rename, Culture, 55 images) should be validated on staging before production.
